Data Specification for Journey Time Indicators
JOURNEY TIME INDICATORS
Journey Time Indicators (JTI) installed on major routes on Hong Kong Island, in Kownloon and in New Terriortories to provide the latest journey time for 
motorists to make an informed route choice.  The displayed journey time refers to the average journey time of vehicles from the JTI to the respective
destinations.
The journey time XML file is updated every 2 minutes.
Data structure of the XML file for JTI is as follows:
